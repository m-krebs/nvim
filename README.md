# Neovim Configuration

To use this nvim configuration link(-s) this directory from the default nvim configuration directory.
Usually this is at following locations: 

| Linux/MacOS | Windows |
| --- | --- |
| `~/.config/nvim` | `~/AppData/Local/nvim` |

Using the command `:echo stdpath('config')` inside of neovim to show the directory.
